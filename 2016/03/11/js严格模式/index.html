js严格模式

javascript 严格模式

  • 严格模式下，不允许使用 with 语句。只有在执行时才可以确定对象的具体属性，而严格模式在编译时就检查变量，故语法错误。
  • 严格模式下，this 默认为 undefined。
    • 若忘记了使用 new 运算符，则会抛出错误。

  • 禁止在函数内部遍历调用栈。

  • 定义重复的属性或重复的参数名。

  • 不可以对禁止扩展的对象添加新属性。

  • 不可删除一个不可删除的属性。

  • 严格模式下，不允许对 arguments 对象赋值，arguments 不再追踪参数的变化。

  • 函数声明必须在顶层。
